Installation

- Extract the contents of statcato.zip to a 
desired location.  
- To launch Statcato, open the file
statcato.jar
- All the java source files are in the src folder.
- Library files used in this application are found
in the lib folder.
- Documentation files are in the javadoc folder.

* Statcato requires Java Runtime Environment (JRE) 
1.6 or higher. To download the latest version of Java, 
visit http://www.java.com/en/download/index.jsp.
