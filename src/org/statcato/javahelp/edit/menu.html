Edit Menu

The edit menu provides functions for editing Datasheets.
  • Cut: Cut selected contents to the clipboard.
  • Copy: Copy selected contents to the clipboard.
  • Paste: Paste contents from the clipboard to the insertion point.
  • Select All: Select all cells in the current Datasheet
  • Clear Cells: Clear the contents in the selected cells
  • Delete Cells: Delete the selected cells. Cells below the deleted cells are shifted up to replace the deleted ones. The size of the Datasheet is kept unchanged.
  • Insert Rows: Choose Above to insert a blank row above the current cell. Choose Below to insert a blank row below the current cell.
  • Insert Columns: Choose Left to insert a blank column to the left of the current cell. Choose Right to insert a blank column to the right of the current cell.
  • Insert Cell: Choose Above to insert a blank cell above the current cell. Choose Below to insert a blank cell below the curent cell. Cells are shifted up/down to accommodate the inserted cell.
  • Add Multiple Rows/Columns: Add a user-specified number of rows and columns to the end of the spreadsheet.
