Graph > Residual Plot

The residual data of linear regression is the difference between the observed data of a variable (y) and the fitted / predicted values of that variable (y').

The residual plot features are available in the Linear Correlation and Regression utility. Five residual plot options are available:

  • Residuals vs. x variable
  • Residuals vs. predicted (fitted) values
  • Normal probability plot of residuals
  • Histogram of residuals
  • Residuals vs. observation order

To create residual plots, check the Show Residual Plots checkbox, and then select the desired plot options. If multiple residual plots are selected, the plots will be shown in a combined plot, with two residual plots in a row of the combined plot. Other dialog inputs are discussed in the Linear Correlation and Regression section.
