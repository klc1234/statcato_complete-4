Calculate > Probability Distributions > Chi-square

This utility computes the probability density, cumulative probability, and inverse cumulative probability of the Chi-square distribution.

Distribution Parameters:

  • Specify the degrees of freedom.

Compute:

  • Select the desired type of computation.

Input(s):

  • If the input data is in a column in the current Datasheet, select the Column radio button and the column name in the drop-down menu.
  • In the input data is a single constant, select the Constant radio button and enter the input value in the provided text box.

Store Results in: (optional)

  • Computed results can be stored in a column of the current Datasheet. If storing results is desired, enter the column name or variable name in the provided text box.
