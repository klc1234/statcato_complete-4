Data > Generate Random Data > Uniform

This utility simulates the process of drawing random samples from a range of uniformly distributed continuous values. To use this utility, you must specify the column name(s) in which samples will be stored, the sample size, and the minimum and maximum values of the range of possible values. Providing a seed for the random number generator is optional.

Dialog Inputs

  • In the Store Samples in: text field, enter the column names in which random samples will be stored, separated by spaces. For a continuous range of columns, specify using the first column, a dash, and the last column (e.g. C1-C30).
  • In the Number of Samples to Generate: text box, enter the number of random samples to generate for each column.
  • Enter the minimum value (inclusive) to generate in the Minimum: text field.
  • Enter the maximum value (exclusive) to generate in the Maximum: text field.
  • (optional) Enter a seed for the random number generator in the Random Generator Seed: text box.
