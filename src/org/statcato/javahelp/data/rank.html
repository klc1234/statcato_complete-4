Data > Rank

The rank utility assigns ranks to numerical values in a column. The ranks are computed as follows:

  • The values are first sorted in ascending order.
  • The smallest value is assigned a rank of 1. The next smallest value in the sorted list is assigned a rank of 2, and so on.
  • The rank of repeated values is the average of their corresponding positions in the sorted list of values.

To use the utility, select Data > Rank. In the rank dialog, select the column containing data to be ranked using the Rank Data in: dropdown menu. Enter the column or variable name for storing ranks in the Store Ranks in: text box.
