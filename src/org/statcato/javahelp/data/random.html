Data > Generate Random Data

This menu contains utilities for generating random data:

  • Binomial: Generate samples from a binomial distribution given the number of trials and the event probability.
  • Integer: Generate samples from a range of integers, each of which has the same probability of being chosen.
  • Normal: Generate samples from a normal distribution.
  • Uniform: Generate samples from a uniformly distributed range of continuous values.
  • Samples from Column: Generate samples from a given column of data values. The option of random sampling with or without replacement is provided.
