Statistics > Correlation and Regression > Linear Two Variables

This utility performs computations that determine the correlation and regression between two variables. To use this utility, you must provide the paired x (independent/predictor) and y (dependent/response) values in separate columns. You must also provide the significance level for the hypothesis test for correlation.

Dialog Inputs

  • In the Independent/dependent variable series list, select the columns containing data values for the input variables. The x variable is the independent variable, and the y variable is the dependent variable. Click Clear Input List to clear the input columns list.
  • Significance level: Enter the significance level of the hypothesis test for correlation.
  • Select the Show a scatterplot for all pairs of data values check box to display a scatter plot showing data values in each pair of input variables. Use the provided options to customize the scatterplot.
