Statistics > Correlation and Regression > Correlation Matrix

This utility allows you to specify a list of variables and performs linear correlation and regression for each pair of variables. The results are displayed for each pair of variables in a matrix.

To use this utility, provide at least two variables in separate columns. Also provide the significance level for the hypothesis test for correlation.

Dialog Inputs

  • To add a variable to the Variable list, select the desired column from the Column list and then click << Add to list. To remove a variable from the Variable list, select the column and then click > Remove from list >>. Click Clear Input List to clear the input columns list.
  • Significance level: Enter the significance level of the hypothesis test for correlation.
  • Select the Show a scatterplot for all pairs of data values check box to display a scatter plot showing data values in each pair of input variables. Use the provided options to customize the scatterplot.
